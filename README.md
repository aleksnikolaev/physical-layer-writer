# Physical Layer Writer

Claude Code skill for writing investigative analysis posts. Calibrated for one specific voice and domain — physical-layer constraints (water, energy, land) — with fact-check pipeline, per-claim risk scoring, and a fixed post structure (Map → Signals → Strategy Patches → Limits → Manipulation Analysis).

Adaptation example for [founder-voice-ghostwriter](https://github.com/BayramAnnakov/founder-voice-ghostwriter) by Bayram Annakov. The structural patterns are reusable; the voice and domain calibration are not.

Output archive: [Physical Layer on Substack](https://alexnik2.substack.com/)
